# Note-Taker

## Description

This is an application that writes and saves notes to organize and keep track of tasks. When you click on the link to the notes page you are presented with a page with existing notes listed in the left-hand column, plus empty fields to enter a new note title and the noteâ€™s text in the right-hand column.
